# Institutional Request Manager – Power Platform Project

## Project Overview

This project demonstrates a low-code institutional workflow solution using Microsoft Power Platform.

The application allows internal users to:

- Submit structured requests
- Track request status
- Trigger automated notifications
- Improve operational efficiency

---

## Architecture

- PowerApps Canvas Application
- SharePoint Online (Data storage)
- Power Automate (Workflow automation)
